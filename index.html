This app records positions of the phone at a user-defined frequency and monitors the phone accelerometers keeping track of shocks to the smartphone that might indicate problems to the carrier.
The coordinates of the positions are sent directly to a server to track in real time the movements of the phone; this option can be turned off by the user, and is automatically turned off if not network is found.
The locations are saved locally when user stops to record positions, and all positions are sent to server and saved locally. If sending to server is successful then the local copy is deleted. If server connection fails (e.g. no network or server is down) then the local copy is stored and synced at later time. Syncing can be forced by the user, or it is done automatically at next app launch.
Location data are downloaded in geoJson format (compressed in a ZIP archive), compatible with all GIS software, e.g. QGIS.
The big red button can be used to send a signal to the server and trigger further actions (e.g. emails or calls).
